# Skills Learned

## Overview

The Certified Computer Forensics Analyst (CCFA) certification enhanced my understanding of digital forensic investigation methodologies, evidence handling procedures, forensic analysis techniques, and cybersecurity fundamentals.

---

## Technical Skills Acquired

### Digital Evidence Handling
- Identifying, collecting, preserving, and documenting digital evidence while maintaining its integrity.

### Chain of Custody
- Understanding the proper procedures for handling and documenting evidence to ensure legal admissibility.

### Evidence Acquisition
- Creating forensic copies (disk imaging) without modifying the original evidence.

### Windows Forensics
- Investigating Windows artifacts including:
  - Registry
  - Prefetch Files
  - ShellBags
  - LNK Files
  - Alternate Data Streams (ADS)

### Linux Forensics
- Understanding Linux file systems and forensic artifacts.

### Mobile Forensics
- Collecting and analyzing evidence from Android and iOS devices.

### Network Forensics
- Capturing and analyzing network traffic using packet capture (PCAP) files and Wireshark.

### File System Analysis
- Understanding file headers, metadata, magic numbers, and common file systems.

### Cryptography Fundamentals
- Learning symmetric and asymmetric encryption concepts, hashing algorithms, AES, RSA, and ECC.

### Incident Response
- Applying digital forensic techniques during cybersecurity investigations.

---

## Outcome

This certification strengthened my practical understanding of Digital Forensics and Incident Response (DFIR) concepts and improved my ability to investigate digital evidence using industry-recognized methodologies.
